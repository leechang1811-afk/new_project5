# 오늘 1개만 완료 QA 체크리스트 (v1)

아래 20개를 모두 통과하면 "판매 가능한 기본 품질"로 판단한다.

1. 앱 첫 진입 시 3초 사용법 모달이 뜬다.
2. 사용법 모달 `바로 시작` 탭 시 즉시 닫힌다.
3. 체크인에서 할 일 미입력 시 `시작하기`가 비활성화된다.
4. 체크인 완료 후 체크아웃 화면으로 전환된다.
5. 체크아웃에서 `✅ 다 했어요` 탭 시 바로 저장 플로우가 실행된다.
6. 체크아웃에서 `😓 아직 못했어요` 선택 시 이유 선택 UI가 노출된다.
7. 미완료 상태에서 이유 미선택 시 저장 버튼이 비활성화된다.
8. 같은 날 저장 성공 후 다시 저장 시 중복 저장이 차단된다.
9. 중복 저장 차단 시 안내 토스트가 노출된다.
10. 완료 저장 후 WOW 모달이 노출된다.
11. WOW 모달에 점수/연속일/주간 수치가 표시된다.
12. 완료 저장 시 컨페티가 충분히 강하게 보인다.
13. 설정 패널에 카테고리/시간이 "선택"임이 보인다.
14. 시간 설정이 점수와 무관하다는 문구가 보인다.
15. 주간 리포트 진입/복귀 버튼이 정상 동작한다.
16. 공유 문구 복사 버튼 탭 시 성공/실패 토스트가 정상 표시된다.
17. 새로고침 후에도 목표/할 일/기록/시간이 유지된다.
18. 모바일 폭(375px)에서 상단/하단 UI가 잘리지 않는다.
19. 태블릿 폭(768px 이상)에서 중앙 정렬/가독성이 유지된다.
20. 데스크톱 폭에서 CTA 버튼과 핵심 텍스트가 즉시 식별된다.
