# 퇴근력 UX Wireflow + 화면 카피

## 화면 1) 온보딩
### 목적
- 사용자 목표 맥락을 빠르게 고정하고, 2회 리마인드 시간을 확보한다.

### 구성
- 타이틀: "하루 2번, 실행률을 올려보세요"
- 목표 선택 카드: 업무 / 건강 / 학습 / 관계
- 알림 설정: 아침 시간, 저녁 시간
- CTA: "내 루틴 시작하기"

### 마이크로카피
- 서브카피: "긴 계획 대신, 오늘 할 1개만 정합니다."
- 허용 안내: "언제든 설정에서 바꿀 수 있어요."

## 화면 2) 아침 체크인 (30초)
### 목적
- 오늘의 1개를 명확히 선택한다.

### 구성
- 헤더: "출근 전 30초 체크인"
- 추천 액션 3개 (카테고리 기반)
- 직접 입력 1줄
- CTA: "오늘의 1개 확정"

### 마이크로카피
- 가이드: "완벽한 목표보다, 끝낼 수 있는 목표가 좋아요."
- 완료 토스트: "좋아요. 오늘은 이 1개만 끝내요."

## 화면 3) 저녁 체크아웃 (60초)
### 목적
- 실행 결과를 입력하고, 실패 시 이유를 1탭으로 남긴다.

### 구성
- 헤더: "퇴근 후 60초 체크아웃"
- 질문: "오늘의 1개, 완료했나요?"
- 버튼: 완료했어요 / 못했어요
- 미완료 선택 시 이유 태그: 시간부족 / 피곤 / 우선순위 밀림 / 예상보다 어려움
- CTA: "결과 보기"

### 마이크로카피
- 완료 시: "좋아요. 오늘도 실행을 지켰어요."
- 미완료 시: "괜찮아요. 내일은 더 작게 시작하면 됩니다."

## 화면 4) 일일 결과 카드
### 목적
- 성취를 즉시 수치로 전달한다.

### 구성
- 실행 점수 (예: 72점)
- 연속 달성일 (streak)
- 이번 주 실행률
- 전주 대비 개선률
- CTA: "내일도 이어가기"

### 마이크로카피
- 상승 구간: "지난주보다 +8% 좋아졌어요."
- 하락 구간: "내일 1개만 완료하면 다시 반등합니다."

## 화면 5) 주간 리포트/배지
### 목적
- 재방문 이유를 강화하고, 성취의 누적 경험을 제공한다.

### 구성
- 주간 히트맵 (요일별 완료 여부)
- 잘된 패턴 1개 / 막힌 패턴 1개
- 새 배지 획득 영역
- CTA: "다음 주 목표 유지"

### 마이크로카피
- "당신은 화/목 저녁에 가장 강해요."
- "월요일은 목표를 더 작게 잡으면 유지율이 올라갑니다."

## UX 원칙
- 1스크린 1의사결정
- 실패를 벌점이 아닌 복구 신호로 처리
- 광고/결과/CTA 레이어 충돌 금지
- 텍스트보다 선택형 입력 우선
